# GROOT Brigata — Opensource Profile

This is a sanitized, opensource-ready version of the **GROOT Brigata** Hermes profile.

All sensitive/private information has been replaced with placeholders:
- `{{HERMES_HOME}}` — profile root directory
- `{{TELEGRAM_CHAT_ID}}` — Telegram chat IDs
- `{{TELEGRAM_GROUP_CHAT_ID}}` — Telegram group chat IDs
- `{{TELEGRAM_ADMIN_CHAT_ID}}` — Telegram admin chat IDs
- `{{TELEGRAM_USER_ID}}` — Telegram user IDs
- `{{TELEGRAM_THREAD_ID}}` — Telegram thread IDs
- `{{GOOGLE_SHEETS_ID}}` — Google Sheets spreadsheet IDs
- `{{UPDATED_AT}}` — timestamps
